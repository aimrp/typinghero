# 打字英雄 (Typing Hero)

一款基于 Electron 开发的趣味打字练习游戏，结合了即时反馈、语音嘲讽/鼓励和成就系统，旨在让枯燥的打字练习变得生动有趣。

## 🎮 游戏特色

*   **实时语音反馈**：
    *   **毒舌嘲讽**：当你打字慢吞吞时，会有御姐音对你进行花式嘲讽（"才10秒？键盘都还没热身呢！"）。
    *   **暖心鼓励**：当你完成一段输入时，会有真挚的语音鼓励。
    *   **时间提醒**：在 10s, 20s, 30s, 60s 等关键节点，会有专门的语音提示，催促你加快速度。
*   **丰富的视觉特效**：
    *   **Combo 系统**：连击数达到一定程度会触发火焰特效。
    *   **粒子爆炸**：打字、完成、胜利时都有炫酷的粒子爆炸效果。
    *   **动态背景**：根据游戏进度和状态变化的界面元素。
*   **经典素材库**：
    *   内置大量唐诗、宋词、毛主席诗词以及经典古文名句，让你在练习打字的同时感受中华文化的魅力。
*   **进阶音乐系统**：
    *   随着连击数 (Combo) 的提升，背景音乐会自动切换，从舒缓的圆舞曲到激昂的土耳其进行曲，再到悲怆奏鸣曲，情绪层层递进。
*   **跨平台支持**：
    *   支持 macOS 和 Windows 系统，安装包体积小巧，安装即用。

## 🛠️ 技术栈

*   **Electron**: 构建跨平台桌面应用。
*   **HTML/CSS/JS**: 原生前端技术栈，无复杂框架依赖，轻量高效。
*   **Minimax API**: 用于生成高质量的语音合成 (TTS) 素材。

## 🚀 快速开始

### 开发环境运行

1.  克隆项目到本地：
    ```bash
    git clone https://github.com/YOUR_USERNAME/typinghero.git
    cd typinghero
    ```

2.  安装依赖：
    ```bash
    npm install
    ```

3.  启动应用：
    ```bash
    npm start
    ```

### 打包构建

*   **构建 macOS 安装包 (.dmg)**:
    ```bash
    npm run dist -- --mac
    ```

*   **构建 Windows 安装包 (.exe)**:
    ```bash
    npm run dist -- --win
    ```

## 📂 项目结构

*   `main.js`: Electron 主进程入口，负责窗口管理和系统交互。
*   `keyboard.html`: 游戏主界面及核心逻辑。
*   `audio_assets/`: 存放生成的语音文件 (嘲讽、鼓励、失败等)。
*   `audio_bgm/`: 存放背景音乐文件。
*   `scripts/`: 辅助脚本目录。
